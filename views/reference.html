Okay...what does this website do?

This is simply an API that serves data scraped from RDS Offered Courses List of North South University in JSON format. And we all love JSON, right?



API References

The API endpoints so far are:


Route Parameters Example What it gives back
/api/courses None /api/courses Gives back the entire list of courses.
/api/courses/:coursecode :coursecode /api/courses/cse465 Given a course code, returns all the sections offered for it.
/api/notfilledcourses/:timing :timing /api/notfilledcourses/st940
/api/notfilledcourses/mw1120
/api/notfilledcourses/s800 		Given a time, returns the not filled up courses with that timing.
/api/notfilledcourses/:course :course /api/notfilledcourses/cse465
/api/notfilledcourses/env203
/api/notfilledcourses/act201 		Given a course code, returns the not filled up sections of that course.

Okay...What are the bugs?

Randomly, the python scrapper will fail to dump the entire JSON content into the database. In that case, obviously, the JSON parser call will fail. Simply wait for a minute before retrying.
